# NetFlux

## Running / Development

* Database with MongoDB

  * `sudo mongod`
  
* Server

 * `cd NetFloux/serveur`
 
 * `npm install`
 
 * `node app`

* Client 

 * `cd NetFloux/client`
  
 * `npm install`
  
 * `webpack-dev-server`
  
 * Visit the app at [http://localhost:8080](http://localhost:8080/Home)
